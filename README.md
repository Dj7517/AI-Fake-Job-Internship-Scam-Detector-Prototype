# AI Fake Job & Internship Scam Detection System (Prototype)

## 📖 Project Overview
The AI Fake Job & Internship Scam Detection System is a prototype-based solution designed to help students identify fraudulent job and internship offers received through digital platforms such as WhatsApp, email, LinkedIn, and Telegram. With the increasing number of online scams targeting students, there is a strong need for an intelligent and reliable verification mechanism.

This project demonstrates how Artificial Intelligence can be used conceptually to analyze message content and warn users before they fall victim to scams.

---

## 🚨 Problem Statement
Students frequently receive fake job and internship offers that promise guaranteed placements, high salaries, or paid internships. These scam messages often request registration fees, urgent responses, or sensitive personal information. Due to the lack of a quick and trusted verification tool, many students suffer financial loss, data misuse, and emotional stress.

---

## 💡 Proposed Solution (Prototype)
This project presents a prototype of an AI-powered system where users can paste job or internship messages into a web-based interface. The system conceptually uses Google Gemini API to analyze the text using Natural Language Processing (NLP) and detect scam indicators such as urgency, unrealistic promises, payment requests, and missing company details.

Based on the analysis, the system classifies the message as either **Scam** or **Genuine** and provides a brief explanation to educate users and promote safe decision-making.

---

## 👥 Target Users
- College students

---

## 🔄 User Flow
1. User pastes a job or internship message  
2. Message is analyzed using Google Gemini API  
3. Scam indicators are identified  
4. The system classifies the message  
5. Result and safety warning are displayed
